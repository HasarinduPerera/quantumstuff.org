Accessibility Statement

Our commitment to making quantum education accessible to everyone

Our Commitment

quantumstuff.org is committed to ensuring digital accessibility for people with disabilities. We are continually improving the user experience for everyone and applying the relevant accessibility standards.

We believe that everyone should have access to quality quantum computing education, regardless of their abilities or the technologies they use to access the web.

Conformance Status

WCAG 2.1 Level AA - quantumstuff.org aims to conform to the Web Content Accessibility Guidelines (WCAG) 2.1, Level AA. We are actively working towards full compliance.

These guidelines explain how to make web content more accessible for people with disabilities and user-friendly for everyone.

Accessibility Features

Our website includes the following accessibility features:

Theme Options: Light, Dark, and High Contrast modes
Font Size Controls: Adjustable text size (A-, A, A+)
Dyslexia-Friendly Font: OpenDyslexic option available
Text-to-Speech: Read page aloud using Web Speech API
Keyboard Navigation: Full site navigation via keyboard
Skip Links: Skip to main content option available
Semantic HTML: Proper landmarks and structure for screen readers
ARIA Labels: Descriptive labels for assistive technologies

Keyboard Shortcuts

Action Shortcut
Navigate between elements Tab / Shift+Tab
Activate link or button Enter or Space
Skip to main content Tab (first on page)
Scroll page Arrow Keys, Page Up/Down

Known Issues

We are aware of the following accessibility issues and are actively working to resolve them:

  • Some interactive visualizations may not be fully keyboard accessible
  • Video content does not yet have captions (we will add them when video content is created)
  • Some color contrasts are still being optimized for WCAG AAA compliance

Feedback

We welcome your feedback on the accessibility of quantumstuff.org. Please let us know if you encounter any accessibility barriers.

Contact Methods:

  • Email: accessibility@quantumstuff.org
  • GitHub: Report an issue

We try to respond to accessibility feedback within 2 business days.
